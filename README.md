## Getting Started
- Clone the repo and cd into the directory
```sh
$ git clone https://github.com/raj713335/My_Resume_Flask.git
$ cd My_Resume_Flask
```

- Install Flask

```sh
$ pip install Flask
```

- Run the app

```sh
$ python app.py
```
